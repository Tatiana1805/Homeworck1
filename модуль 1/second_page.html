Вернуться на главную страницу

HTML

  • Отвечает за структуру элементов
  • В HTML встречаются такие теги, как: div, span, button, img и другие
  • Отвечает за текстовый и мультемидийный контент на веб-странице

CSS

  • Отвечает за оформление элементов
  • В CSS присутствуют такие свойства, как: color, font-size, widht, padding и другие
  • Отвечает за внешний вид веб-страницы

JavaScript

  • Отвечает за динамику и взаимодействие с сайтом
  • В JavaScript существуют функциональные и классовые компоненты
  • Может изменять содерживое и структуру страницы, а также сохранять и обновлять данные (и не только)
